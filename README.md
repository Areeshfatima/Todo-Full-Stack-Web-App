# Todo Mastery - Professional Todo Web Application

A premium todo application with elegant design and smooth interactions built with Next.js, FastAPI, and modern web technologies.

## Overview

Todo Mastery is a full-stack web application featuring:
- **Frontend**: Next.js 16+ with App Router, TypeScript, Tailwind CSS
- **Backend**: FastAPI with SQLModel and PostgreSQL
- **Authentication**: Better Auth with JWT tokens
- **Database**: Neon PostgreSQL with proper user isolation

## Features

- **Authentication**: Secure signup and login with JWT handling
- **Task Management**: Create, read, update, and delete tasks with optimistic updates
- **Premium UI**: Elegant design with smooth animations and transitions
- **Responsive Design**: Works seamlessly across mobile, tablet, and desktop
- **Accessibility**: Full keyboard navigation and ARIA support
- **User Isolation**: Each user sees only their own tasks

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Python 3.9+ (for backend)
- PostgreSQL (or Neon account)

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env.local` file with the following:
   ```env
   NEXT_PUBLIC_API_BASE_URL=http://localhost:8000/api
   BETTER_AUTH_URL=http://localhost:3000
   BETTER_AUTH_SECRET=your-super-secret-jwt-token-here-change-me
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
