# Frontend Architectural Decisions

## UI Overhaul - Phase 2

### Hero background
- **Decision**: Use a linear-gradient from indigo-50 to white for a clean, premium feel.
- **Rationale**: Provides a modern and elegant aesthetic, aligning with the "professional UI overhaul" goal.

### Form style
- **Decision**: Implement floating labels for form inputs.
- **Rationale**: Offers a modern and elegant user experience, optimizing space and clarity.

### Button style
- **Decision**: Use solid indigo buttons with a subtle shadow and a hover darken effect.
- **Rationale**: Ensures clear call-to-actions with a professional and interactive feel.
